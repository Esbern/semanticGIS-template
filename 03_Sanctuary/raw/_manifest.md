# Raw Data Manifest

This folder holds unmodified source data exactly as downloaded or received.
- No reprojection. No column filtering. No geometry transformation.
- All original attributes and the original CRS are preserved.
- Each file is logged in the table below before any processing begins.

Derived (processed) layers are stored in `03_Sanctuary/processed/`.
The Sanctuary Index (`03_Sanctuary/Sanctuary_Index.md`) records the transformation lineage from raw to processed.
